# Tarefas

Este diretório contém tarefas práticas para colocar em prática os conceitos aprendidos no curso de Fundamentos de Programação.
Cada tarefa é apresentada em um arquivo Markdown separado, contendo o enunciado do problema, a solução em código Python e informações sobre o autor e a data de criação.

As tarefas abrangem uma variedade de tópicos, desde manipulação de arquivos e estruturas de dados até algoritmos matemáticos e jogos simples.

As tarefas neste diretório **não são todas aplicadas em aula**, pois algumas exigem maior tempo de dedicação e têm maior complexidade.


## Recomendação

Abra a tarefa, leia o enunciado com atenção, e elabore sua própria solução em um arquivo Python separado.
Depois, compare sua solução com a solução apresentada no arquivo Markdown.
