## Legenda de Dificuldade

- **Fácil**: Conceitos básicos, ideal para iniciantes
- **Médio**: Requer conhecimento de estruturas de controle e laços
- **Difícil**: Envolve estruturas de dados complexas (vetores, matrizes)
- **Muito Difícil**: Combina múltiplos conceitos avançados

## Dicas de Estudo

1. Comece pelos exercícios mais fáceis e avance gradualmente
2. Tente resolver o exercício antes de consultar a solução
3. Analise a solução fornecida e compare com sua abordagem
4. Experimente variações dos exercícios para fixar o aprendizado
